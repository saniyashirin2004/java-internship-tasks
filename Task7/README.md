# Employee Database App (Java JDBC)

A simple Java application using JDBC to perform CRUD operations on an Employee table.

## Features
- Connect to MySQL/PostgreSQL database
- Add employee
- View employees
- Update employee
- Delete employee

## Tools Used
- Java
- JDBC Driver
- MySQL/PostgreSQL
- VS Code

## How to Run
1. Create database and Employee table.
2. Update DB username/password in the code.
3. Run the Java file to perform CRUD operations.
